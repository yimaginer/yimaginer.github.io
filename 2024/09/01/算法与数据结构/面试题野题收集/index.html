面试题刷题：

字母和数字

https://leetcode.cn/problems/find-longest-subarray-lcci/description/

  • 前缀和+哈希表
  • 前缀和方便处理一段的性质
  • 哈希表方便查找存在性问题
  • 字母和数字的区分用 array[i][0] >> 6 & 1

和为 k 的数组

https://leetcode.cn/problems/subarray-sum-equals-k/

  • cnt 哈希表用来记录前缀和为 first 的点的个数 second
  • ans += cnt[sj - k] 一开始担心会加到位置靠后的前缀和为 sj - k 的点，但是 for (int sj : s) 这样的遍历顺序决定了只会加到前面的点
  • contains 是 C++ 20 才引入的接口，返回 true / false, 同样功能的接口是 count 和 find
