C++ 多态机制的内存管理

一个 C++ 类的四种成员：

  • 非静态成员变量：属于类，参与继承。
  • 静态成员变量：位于全局变量区域，不参与继承，属于类而不属于任何实例。必须类内声明，类外定义。

  • 非静态成员函数：

单继承

  • 虚函数表指针放在最前
  • 成员变量依据继承和声明顺序放在后面
  • 被重写的虚函数在虚函数表中进行更新

多继承

  • 父类按照继承顺序依次排列，每个父类有自己的虚表。
  • 子类的新定义虚函数被放到了第一个父类的表中。
  • 子类重写的虚函数会在对应的父类虚函数表中更新。

出现钻石继承的虚继承的时候，虚基类在子类中只有一份。
出现钻石继承的非虚继承的时候，虚基类在每个子类中都有一份。
